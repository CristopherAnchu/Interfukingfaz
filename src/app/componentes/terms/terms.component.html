1. Aceptación de los Términos

Al acceder y utilizar el Sistema de Gestión Documental de la Universidad Laica Eloy Alfaro de Manabí (ULEAM), usted acepta estar sujeto a estos Términos y Condiciones de Uso. Si no está de acuerdo con alguno de estos términos, por favor no utilice el sistema.

2. Uso del Sistema

Este sistema está diseñado exclusivamente para el personal autorizado de la ULEAM para:

  • Gestionar documentos institucionales
  • Compartir información académica y administrativa
  • Facilitar la comunicación entre departamentos
  • Mantener registros documentales oficiales

3. Responsabilidades del Usuario

Los usuarios se comprometen a:

  • Mantener la confidencialidad de sus credenciales de acceso
  • Utilizar el sistema únicamente para propósitos institucionales
  • No compartir información sensible con personas no autorizadas
  • Notificar inmediatamente cualquier uso no autorizado de su cuenta
  • Respetar los derechos de propiedad intelectual

4. Seguridad y Acceso

El sistema implementa medidas de seguridad incluyendo:

  • Bloqueo temporal de cuenta después de 3 intentos fallidos de inicio de sesión
  • Encriptación de datos sensibles
  • Registro de actividades del usuario
  • Control de acceso basado en roles

5. Contenido y Propiedad Intelectual

Todos los documentos y contenidos cargados al sistema son propiedad de la ULEAM o de sus respectivos autores. Los usuarios no deben copiar, distribuir o modificar contenido sin autorización expresa.

6. Privacidad

La ULEAM se compromete a proteger la privacidad de los usuarios. Para más información, consulte nuestra Política de Privacidad.

7. Limitación de Responsabilidad

La ULEAM no se hace responsable por:

  • Interrupciones del servicio debido a mantenimiento o problemas técnicos
  • Pérdida de datos debido a errores del usuario
  • Uso inadecuado del sistema por parte de usuarios autorizados
  • Daños indirectos o consecuentes derivados del uso del sistema

8. Modificaciones

La ULEAM se reserva el derecho de modificar estos términos en cualquier momento. Los usuarios serán notificados de cambios significativos a través del sistema.

9. Terminación del Acceso

La ULEAM se reserva el derecho de suspender o terminar el acceso de cualquier usuario que viole estos términos o utilice el sistema de manera inapropiada.
